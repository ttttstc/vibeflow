# Retrospective 2026-03-21

## What Worked
- The sample project stayed small enough to move through all workflow stages quickly.
- Template-driven workflow generation reduced manual setup.
- Standard library only implementation kept verification simple.

## What Did Not
- The initial workflow scripts assumed templates lived inside the target project.
- Work-config generation had a PowerShell interpolation bug.

## Next Improvements
- Keep template resolution anchored at the VibeFlow repository by default.
- Add a helper for creating think output from a short prompt.
